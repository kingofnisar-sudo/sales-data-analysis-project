# Sales Data Analysis Project

## Overview
This project focuses on analyzing sales data to extract meaningful insights and support business decision-making using multiple data analytics tools.

## Tools Used
- Microsoft Excel
- SQL
- Python (Pandas, NumPy)
- Power BI

## Project Workflow

### 1. Data Cleaning (Python)
- Cleaned raw dataset using Pandas
- Handled missing values and corrected data formats
- Performed basic calculations using NumPy

### 2. Data Analysis (SQL)
- Imported data into SQL database
- Used queries for filtering, grouping, and aggregation
- Applied functions like JOIN, GROUP BY, and WHERE for analysis

### 3. Data Visualization (Excel & Power BI)
- Created interactive dashboards in Power BI
- Built Excel dashboards using Pivot Tables and charts
- Visualized KPIs such as Revenue, Profit, and Regional Performance

## Key Insights
- Identified top-performing regions based on sales
- Analyzed monthly and category-wise sales trends
- Highlighted low-performing areas for improvement
